# 项目目录结构文档

> 本文档仅描述项目目录结构及各目录/文件的用途，不涉及文件具体内容。

---

## 根目录

```
.
├── .zhou/                          # Agent 运行配置与数据目录
├── Agent.md                        # Agent 基础提示词 / 行为定义文件
├── docs/                           # 项目文档目录
├── pyproject.toml                  # Python 项目配置文件（依赖、构建等）
├── src/                            # 源代码目录
└── test/                           # 测试目录（含测试用的 Agent 配置与数据）
```


## 文件/目录命名约定

| 命名 | 说明 |
|------|------|
| `SKILL.md` / `skill.md` | Skill 定义文件（大小写不统一，两种均存在） |
| `*.toml` | TOML 格式配置文件 |
| `*.json` / `*.jsonl` | JSON 及 JSON Lines 数据文件 |
| `*.sh` | Shell 脚本 |
| `*.py` / `*.pyc` | Python 源文件及编译字节码 |
| `*.md` | Markdown 文档 |
| `__pycache__/` | Python 自动生成的字节码缓存目录 |
| `*.egg-info/` | Python 包元信息目录 |
