仅覆盖 `src/zhou/` 下代码文件，不包含 `.zhou/`、`docs/`、缓存与打包产物。

## 3. 模块职责概览

- `main.py`：CLI 启动、初始化、REPL、单轮对话主编排
- `session.py`：session 状态、turn 结构、turns 持久化与恢复
- `skills.py`：skills 扫描、skill 文档解析、skills prompt 注入
- `tools.py`：tools.toml 解析、MCP 建链、tools/list、tools/call
- `tui.py`：skills/tools 界面与流式输出渲染
- `global_archive.py`：turn 的全局归档 JSONL 写入
- `core/commands.py`：特殊命令解析
- `core/config.py`：配置发现、合并、解析、默认模板
- `core/errors.py`：统一异常定义
- `core/llm.py`：主模型请求、SSE 解析、tool call 回路
- `memory/commands.py`：`/memory` 命令展示
- `memory/enrichment.py`：启发式 tags / memory_candidates 派生
- `memory/manager.py`：memory 检索、写入、分类落库、异步结果回写
- `memory/model.py`：独立 memory model worker 与 JSON 决策生成
- `memory/__init__.py`：memory 包按需导出，降低循环导入

## 6. 图的节点选取原则

总图：放跨文件主函数与枢纽函数。

子流程图：在对应功能里展开更多细粒度函数，例如：
- tools：`request_mcp()`、`read_mcp_message()`、`parse_tools_list()`
- session：`load_turn_records()`、`append_turn_to_storage()`
- memory：`search_memory()`、`write_versioned_memory()`、`build_memory_model_prompt()`
